Introdução

O objetivo deste curso é apresentar os principais conceitos sobre os ataques XSS, assim como discutir os principais desafios técnicos e de pesquisa para o desenvolvimento de aplicações Web mais seguras. Além disso, o minicurso conta com uma prática experimental e educativa para a conscientizaçao do público sobre os ataques XSS na qual os participantes poderão avaliar seus conhecimentos adquiridos. Ao fim do minicurso espera-se que os participantes sejam capazes de (i) compreender como os ataques XSS são explorados em aplicações Web; (ii) diferenciar como ocorrem os diferentes tipos de ataques que exploram a vulnerabilidade XSS; (iii) identificar boas práticas sobre o desenvolvimento de aplicações Web seguras; (iv) assimilar os principais mecanismos e estratégias para a defesa contra ataques XSS.
